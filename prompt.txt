4️⃣ PROJECT FORKING SYSTEM
────────────────────────────

Feature:

Users can fork public projects into their own account.

Button:

"Fork Project"

Flow:

1. User clicks fork

2. System creates new project:
   
   - new project_id
   - user_id = current user
   - copy all metadata

3. Copy filesystem:

From:
/home/auroracraft-owner/project/

To:
/home/auroracraft-user/forked-project/

4. Copy database data:

- project record
- optionally file action history

5. Redirect user to new project workspace

Rules:

- Only logged-in users can fork
- Forked project becomes PRIVATE by default
